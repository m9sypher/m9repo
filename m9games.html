Notices

⚠️ Live Chat Dependency

Every game on this page is powered entirely by Twitch Chat. Game state, floor progress, and outcomes are relayed to the UI in real time by parsing chat messages directly from the channel's live feed — there is no independent game server driving these visuals. Consequently, if the stream is offline, chat is not being read and none of the games above are playable. To participate, commands must be typed directly into Twitch Chat — either through the embedded chatbox on this page or a standard Twitch chat client — no other input method is supported.

🔒 Cookies & Data Handling

The embedded Twitch video and chat player may prompt your browser to store a cookie. This cookie is set and managed exclusively by Twitch to retain your embed session (e.g. staying signed in) so you aren't required to re-authenticate on every visit — it is not used by this project to collect, log, or store any personal or account information. This page and every linked game page are static files hosted on GitHub Pages: no backend server, database, or logging process runs as part of this deployment. As a result, M9 and the M9 Server infrastructure have no access to your Twitch credentials, session data, or chat history through this project — there is simply no server-side process here capable of capturing it.

For The Curious — How It Works

A simplified look at what actually runs on this page and game pages. These snippets are trimmed down for readability, but reflect the real logic: everything reads public, non-authenticated data, and nothing here talks to a private server.

Note on tracking: Blackjack does not require tracking a username at all — it's multiplayer, so there's only ever one game running at a time, and the page simply pulls every event from it. The other games, however, are treated as single-player once you track a name: only game actions belonging to that exact tracked username are displayed. If you track your own name, you'll only see your own run — other viewers' games will not appear unless you specifically track their username instead.

// You type a username — it's kept in your own browser

localStorage.setItem('tracked_name', username);

// Only chat lines from the bot that mention
// this username are used to update the UI
if (msg.author === 'M9_BOT' && msg.text.includes(username)) {
  updateGameUI(msg);
}
Avatar Fetching
// Public lookup only — no login, no token, no private scopes
function fetchAvatar(username) {
  return fetch(`https://decapi.me/twitch/avatar/${username}`)
    .then(res => res.text()); // -> a public image URL
}
