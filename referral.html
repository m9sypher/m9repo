Rules
🔒
One code per person
Your code is created once and never changes. Choose it carefully.
🎟️
One referral per viewer
Each viewer can only apply a code once — ever. It can't be changed after the fact.
🚫
Can't refer yourself
You can't apply your own referral code. It only works if someone else uses it.
⏱️
No instant payouts
The payout only fires when the threshold is met. Drop-ins don't count.
📊
Track your progress
Use !refstats in chat to see your code stats and referred status anytime.
🧹
Mod-only resets
Codes and applied referrals can only be wiped by a mod. There's no self-service undo.
